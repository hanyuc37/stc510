# stc510
Bash project
1. Clone git repository
  git clone "url"
2. move file from Linux system to git
  `explorer.exe`
3. add files to git folder
   `git.add`
4. insert commit information
  `git.commit -m "message"`
5. push to github
  `git push`
  * need github username and access token
6. get the latest version
  `git pull`
